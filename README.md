📊 Análise de Acidentes de Trabalho - Miranda Services
📌 Descrição do Projeto

Este projeto consiste em um site desenvolvido com HTML, CSS e JavaScript que apresenta uma análise estatística sobre acidentes de trabalho em uma empresa fictícia chamada Miranda Services.

O objetivo principal é demonstrar, de forma visual e interativa, dados relacionados à segurança do trabalho, facilitando a compreensão de padrões, riscos e possíveis melhorias.

Um README bem estruturado é essencial para explicar o funcionamento e organização do projeto, servindo como porta de entrada para outros desenvolvedores entenderem o sistema .

🎯 Objetivos
Apresentar dados de acidentes de trabalho
Utilizar gráficos visuais para análise
Demonstrar habilidades em desenvolvimento web
Criar uma interface moderna, responsiva e interativa

🔍 Análise dos Resultados

Abril apresentou o maior número de acidentes
Quedas são o tipo mais comum
Há variações ao longo dos meses, indicando possíveis fatores externos
A média mensal foi de 7,5 acidentes

💡 Conclusões

Necessidade de reforço em segurança física
Treinamentos periódicos são essenciais
Monitoramento contínuo pode reduzir riscos
Investigação de picos de acidentes é fundamental
